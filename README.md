# AI for Finance Hack

Этот хакатон организован Райффайзенбанком и посвящён применению искусственного интеллекта в финансовой сфере. 

[Подробнее на Challenge.com](https://changellenge.com/championships/aiforfinancehack/?utm_source=email&utm_medium=segment&utm_campaign=it#)

## Бизнес-контекст 
Клиенты банков ежедневно задают множество вопросов 
о продуктах и услугах: по вкладам, кредитам, комиссиям, инвестициям, страхованию. Обработка этих запросов через колл-центры и чат-боты требует значительных вычислительных и человеческих ресурсов, а качество зависит от уровня подготовки сотрудников или ограниченности шаблонов ответов. Использование Al-ассистентов может помочь, но важно гарантировать правильность их ответов. 

## Стек технологий: 
Python, LLM, RAG, API 

## Задача 
Вам предстоит разработать ассистента на основе LLM, который использует базу знаний для ответов на вопросы пользователей.
